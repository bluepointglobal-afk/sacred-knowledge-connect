# AFAQ ESG — Client Report Pack (v1)

This folder is designed to be **client-sendable** in <60 seconds.

## Contents
- `REPORT_TEMPLATE_FULL_v1.html` — print-ready template (use browser **Print → Save as PDF**)
- `SAMPLE_OUTPUT_SHORT_v1.html` — 8–12 page sample output
- `SAMPLE_OUTPUT_FULL_v1.html` — 40–50 page sample output (realistic structure + placeholders)
- `ONE_PAGER_WHAT_YOU_GET_v1.md` — sales-call explainer
- `E2E_REPRO_RUNBOOK_v1.md` — how to reproduce for new clients (intake → generation → PDF)

## Export to PDF
Open any HTML file in Chrome → **Print** → Destination: **Save as PDF**
- Paper: A4
- Margins: Default
- Options: Background graphics ON

## Notes
- Numbers are placeholders unless explicitly sourced.
- The report is structured to align with common frameworks (IFRS S1/S2, TCFD, GRI) while staying audit-friendly.
