# Market Loop Checklist (use every sprint)

## Before building
- [ ] Define ICP (country, company size, role)
- [ ] Define pain + urgency trigger
- [ ] Define the paid outcome (what "done" looks like)
- [ ] Define pricing hypothesis (range)
- [ ] Define 10-prospect list source

## During sprint
- [ ] Ship a testable artifact
- [ ] Run outreach (>=10 touches)
- [ ] Book >=2 calls OR collect >=5 written replies

## After sprint
- [ ] Funnel snapshot (visit→signup→activation→pay)
- [ ] 5 insights max (bullets)
- [ ] Decision: double-down / iterate / kill
- [ ] Update roadmap/backlog accordingly
